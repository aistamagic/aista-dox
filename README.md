
# Aista Jekyll Theme

This is a Jekyll theme created for Aista, Ltd, and more specifically [Magic Cloud](https://polterguy.github.io/) to
document our project.


## Installation

Add this line to your Jekyll site's `_config.yml`:

```yaml
theme: aista-jekyll
```
